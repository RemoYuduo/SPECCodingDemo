# 企业内部福利商城 - 前端架构信息

## 目录结构
```
frontend/
├── public/                 # 静态资源
├── src/
│   ├── components/         # 公共组件
│   │   ├── common/         # 通用组件
│   │   ├── layout/         # 布局组件
│   │   └── ui/             # UI组件库
│   ├── views/              # 页面视图
│   │   ├── user/           # 用户相关页面
│   │   │   └── ProfileView.vue  # 用户个人资料页面(占位)
│   │   ├── admin/          # 管理员相关页面
│   │   │   └── DashboardView.vue # 管理员仪表板(占位)
│   │   ├── auth/           # 认证相关页面
│   │   │   └── LoginView.vue     # 登录页面(占位)
│   │   ├── HomeView.vue    # 首页
│   │   └── ConnectivityTest.vue  # 连通性测试页面
│   ├── router/             # 路由配置
│   │   └── index.js        # 路由定义
│   ├── stores/             # 状态管理
│   │   └── auth.js         # 认证状态管理
│   ├── api/                # API接口封装
│   │   └── index.js        # API接口定义
│   ├── utils/              # 工具函数
│   ├── assets/             # 资源文件
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
├── index.html              # HTML模板
├── vite.config.js          # Vite配置
└── package.json            # 依赖配置
```

## 模块外部特性

### 认证模块 (auth/) - 已实现
- 登录页面：提供员工和管理员登录功能，支持用户类型切换
- 登录状态管理：全局状态管理用户登录信息
- 登出功能：用户和管理员页面均提供登出功能
- 测试账号：首页和登录页提供测试账号信息快速填充

### 用户模块 (user/) - 已实现基础功能
- 个人信息页面：查看和编辑个人资料，包含退出登录功能
- 商品浏览页面：展示可兑换商品列表
- 购物车页面：管理选中的商品
- 订单管理页面：查看历史订单和订单详情
- 积分使用页面：显示积分余额和使用记录

### 管理员模块 (admin/) - 已实现基础功能
- 管理员仪表板：显示系统信息和管理员信息，提供系统状态检查和退出登录功能
- 商品管理页面：商品的增删改查
- 商品上架/下架页面：管理商品状态
- 库存管理页面：管理商品库存和库存预警
- 订单管理页面：查看和处理用户订单

### 公共组件 (components/)
- 布局组件：导航栏、侧边栏、页脚等
- 表单组件：各种表单输入组件
- 列表组件：商品列表、订单列表等
- 弹窗组件：确认框、提示框等

### API模块 (api/) - 已实现基础功能
- 用户API：登录、登出、个人信息、订单等接口
- 商品API：商品列表、商品详情等接口
- 管理员API：商品管理、订单管理等接口
- 健康检查API：服务健康检查和数据库状态检查接口

### 状态管理 (store/)
- 用户状态：登录信息、用户资料、积分等
- 购物车状态：选中的商品列表
- 应用状态：全局设置、加载状态等

### 连通性测试模块 (ConnectivityTest.vue) - 已实现
- 服务连通性测试：测试前后端API连接
- 数据库连通性测试：测试数据库连接状态
- 登录功能测试：测试用户和管理员登录
- API接口测试：测试各类API接口连通性
- 测试结果展示：实时展示测试结果和响应数据
- 测试账号填充：提供测试账号快速填充功能
- 测试商品接口：获取模拟商品数据

### 路由模块 (router/) - 已实现
- 首页路由：应用主页，提供登录入口
- 连通性测试路由：连通性测试页面
- 登录页面路由：用户和管理员登录页面，支持类型切换
- 用户页面路由：用户个人资料页面
- 管理员页面路由：管理员仪表板页面
