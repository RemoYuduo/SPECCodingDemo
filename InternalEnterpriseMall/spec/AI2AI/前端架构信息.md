## 模块外部特性

### 1. 认证模块 (auth)
- **LoginView.vue**: 用户登录页面
  - 功能: 用户名/密码登录、快速填充测试账号(user1/password1, user2/password2, user3/password3, admin/admin123)、回车键登录
  - 状态: 已实现基础功能
  - 依赖: auth.js(store), auth.js(api)
- **RegisterView.vue**: 用户注册页面
  - 功能: 新用户注册
  - 状态: 未实现
  - 依赖: auth.js(store), auth.js(api)

### 2. 用户模块 (user)
- **ProfileView.vue**: 个人信息页面
  - 功能: 个人信息查看与编辑(用户ID、用户名、邮箱、手机、积分、角色、注册时间)、头像上传、密码修改
  - 状态: 已实现基础功能
  - 依赖: user.js(store), user.js(api)
- **PointsView.vue**: 积分管理页面
  - 功能: 积分余额查询、积分使用记录(兑换历史、消费时间、商品信息)、积分获取记录(签到奖励、活动奖励等)、积分兑换规则说明、积分预警提醒
  - 状态: 未实现
  - 依赖: user.js(store), user.js(api)
- **AddressView.vue**: 收货地址管理页面
  - 功能: 收货地址列表、收货地址增删改、默认地址设置、地址选择功能
  - 状态: 未实现
  - 依赖: user.js(store), user.js(api)

### 3. 商品模块 (product)
- **ProductListView.vue**: 商品列表页面
  - 功能: 商品分类浏览(电子、生活、办公用品等)、商品列表展示(商品图片、名称、所需积分、库存状态)、商品排序(按积分、销量、上架时间)、商品搜索(按名称、类别、价格区间)
  - 状态: 未实现
  - 依赖: product.js(store), product.js(api)
- **ProductDetailView.vue**: 商品详情页面
  - 功能: 商品详细描述、规格参数、库存数量、兑换说明、商品收藏
  - 状态: 未实现
  - 依赖: product.js(store), product.js(api)
- **SearchView.vue**: 搜索结果页面
  - 功能: 搜索结果展示、筛选功能
  - 状态: 未实现
  - 依赖: product.js(store), product.js(api)

### 4. 购物车模块 (cart)
- **CartView.vue**: 购物车页面
  - 功能: 添加商品到购物车、购物车商品列表、购物车商品数量调整、购物车商品批量操作(全选、批量删除)、购物车积分合计、购物车数据持久化

### 5. 订单模块 (order)
