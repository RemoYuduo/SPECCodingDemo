# 企业内部福利商城 - 前端架构信息

## 目录结构
```
frontend/
├── public/                 # 静态资源
├── src/
│   ├── components/         # 公共组件
│   │   ├── common/         # 通用组件
│   │   ├── layout/         # 布局组件
│   │   └── ui/             # UI组件库
│   ├── views/              # 页面视图
│   │   ├── user/           # 用户相关页面
│   │   ├── admin/          # 管理员相关页面
│   │   └── auth/           # 认证相关页面
│   ├── router/             # 路由配置
│   ├── store/              # 状态管理
│   ├── api/                # API接口封装
│   ├── utils/              # 工具函数
│   ├── assets/             # 资源文件
│   └── App.vue             # 根组件
├── package.json
└── vite.config.js
```

## 模块外部特性

### 认证模块 (auth/)
- 登录页面：提供员工和管理员登录功能
- 登录状态管理：全局状态管理用户登录信息

### 用户模块 (user/)
- 个人信息页面：查看和编辑个人资料
- 商品浏览页面：展示可兑换商品列表
- 购物车页面：管理选中的商品
- 订单管理页面：查看历史订单和订单详情
- 积分使用页面：显示积分余额和使用记录

### 管理员模块 (admin/)
- 商品管理页面：商品的增删改查
- 商品上架/下架页面：管理商品状态
- 库存管理页面：管理商品库存和库存预警
- 订单管理页面：查看和处理用户订单

### 公共组件 (components/)
- 布局组件：导航栏、侧边栏、页脚等
- 表单组件：各种表单输入组件
- 列表组件：商品列表、订单列表等
- 弹窗组件：确认框、提示框等

### API模块 (api/)
- 用户API：登录、个人信息、订单等接口
- 商品API：商品列表、商品详情等接口
- 管理员API：商品管理、订单管理等接口

### 状态管理 (store/)
- 用户状态：登录信息、用户资料、积分等
- 购物车状态：选中的商品列表
- 应用状态：全局设置、加载状态等
