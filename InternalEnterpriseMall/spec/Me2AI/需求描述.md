# 企业内部福利商城 Me2AI

## 项目概述
本项目旨在开发一个企业内部福利商城系统，为企业员工提供内部福利商品的兑换、购买和管理服务。

## 核心需求

### 1. 用户前端
- 员工账户登录
- 个人信息编辑
- 商品浏览和购物车
- 使用积分下单
- 订单管理


### 2. 管理前端
- 员工账户增删改，支持企业SSO系统集成（OAuth2）
- 商品增删改
- 商品上架/下架状态管理
- 商品库存和库存预警


## 技术架构

### 1. 前端架构
- 使用Vue 3框架
- 组件化开发，提高代码复用性
- 响应式设计，适配PC和移动端
- 使用Pinia进行状态管理

### 2. 后端架构
- 采用微服务架构
- 核心服务拆分：用户服务、商品服务、订单服务、支付服务、积分服务
- Spring Cloud Gateway统一管理请求路由
- 服务间通信采用RESTful API

### 3. 数据存储
- 关系型数据库：MySQL（核心业务数据）
- 缓存：Redis（会话管理、热点数据）
- 对象存储：MinIO（商品图片）

### 4. 集成方案
- 企业SSO系统集成（OAuth2）
- 企业HR系统数据同步（员工信息、部门信息）
- 企业支付系统集成

## 核心数据库设计

### 1. 用户相关
- 用户表（与企业用户表关联）
- 部门表
- 积分账户表
- 积分变动记录表

### 2. 商品相关
- 商品分类表
- 商品信息表
- 商品库存表

### 3. 订单相关
- 订单表
- 订单商品表
- 购物车表

### 4. 支付相关
- 支付记录表
- 积分使用记录表
